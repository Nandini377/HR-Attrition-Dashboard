# HR-Attrition-Dashboard

Project Overview:

This project analyzes employee attrition data to identify key factors influencing employee turnover.
An interactive dashboard was created using Microsoft Excel to support data-driven HR decision-making.

Objectives:
- Analyze overall attrition rate.
- Identify attrition by department, job role, age, and gender.
- Understand trends and patterns in employee turnover.

Tools Used:
 Microsoft Excel 
- Pivot Tables
- Charts & Visualizations

Key Insights:
- Overall attrition rate is 16%.
- Higher attrition observed in certain departments and job roles.
- Employees who travel rarely have higher attrition.

Conclusion:

The dashboard highlights key factors affecting employee retention and helps organizations make informed, data-driven HR decisions.
